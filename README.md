# Embedded Software Development in C/C++ - Course Template

Template repository for the Embedded C/C++ Development course.

## Course Information

- **Duration:** 2 months (180 academic hours)
- **Platforms:** STM32, ESP32, RP2040, Teensy
- **Tools:** VSCode + PlatformIO, ESP-IDF, STM32CubeIDE

📄 **[Full Course Syllabus (PDF)](docs/Embedded%20Software%20Development%20in%20C_C++.pdf)**
